# Area-of-Interest Workflows for Manila Observatory's 

Containerized data processing workflows for Synthetic Aperture Radar (SAR) and optical imagery. The following sensors and platforms are supported:

1. Landsat 8 (Optical)
2. Sentinel2 (Optical)
2. ALOS2-PALSAR2 (SAR)
3. Sentinel1b (SAR)
